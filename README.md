## 📋 Description

**RentCar** est une application web de gestion de location de voitures développée avec **Spring Boot** et **Thymeleaf**.  
Elle permet de gérer les véhicules disponibles, les clients et les réservations de manière simple et intuitive.

---

## ✨ Fonctionnalités

- 🚘 **Gestion des voitures** — Ajouter, modifier, supprimer et consulter les véhicules disponibles
- 👤 **Gestion des clients** — Enregistrement et suivi des clients
- 📅 **Réservations** — Créer et gérer les locations
- 🖥️ **Interface web** — Interface utilisateur responsive avec Thymeleaf

---

## 🛠️ Technologies utilisées

| Technologie | Rôle |
|---|---|
| Java | Langage principal |
| Spring Boot | Framework backend |
| Thymeleaf | Moteur de templates (vues HTML) |
| Spring MVC | Architecture MVC |
| Maven | Gestion des dépendances |
| HTML / CSS | Interface utilisateur |

## 🚀 Lancer le projet

### Prérequis
- Java 17+
- Maven 3.x

### Installation

```bash
# Cloner le dépôt
git clone https://github.com/IbtissamIraoui/mini_projet.git

# Accéder au dossier
cd mini_projet

# Lancer l'application
./mvnw spring-boot:run
```

L'application sera accessible sur : **http://localhost:8080**
